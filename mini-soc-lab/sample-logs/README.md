# Sample Logs

Every event in this directory is synthetic and generated by
`scripts/generate_sample_logs.py`.

The samples:

- Use private RFC 1918 lab addresses
- Use fictional host and account names
- Contain no credentials or tokens
- Contain no malware or executable payloads
- Are intentionally small enough for code review

Regenerate and evaluate them:

```bash
python scripts/generate_sample_logs.py
python scripts/evaluate_detections.py
```

Do not replace these with unsanitized production logs. Place temporary live captures in the
gitignored `live-evidence/` directory, sanitize selected excerpts, and document the process.
