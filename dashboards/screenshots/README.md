# Dashboard Images

`soc-overview.svg` is a **design mockup**, not a screenshot of a running Wazuh deployment.
Its visible subtitle states this explicitly.

After building the live lab:

1. Create panels from `dashboard-spec.json`.
2. Capture a screenshot with the time range and environment visible.
3. Remove usernames, tokens, public addresses, and unrelated events.
4. Save it under `dashboards/screenshots/live/`.
5. Record the Wazuh version and capture date in the pull request or commit.
