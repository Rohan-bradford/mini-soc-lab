- [ ] Install current security updates.
- [ ] Enable Windows Defender and tamper protection.
- [ ] Configure Windows Firewall for the lab subnet only.
- [ ] Use separate standard and administrative accounts.

- [ ] Disable unused remote services.
- [ ] Configure account lockout.
- [ ] Protect Wazuh agent enrollment material.
